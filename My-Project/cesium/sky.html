sky 天空盒子 ​

在cesium当中可以通过viewer.value.scene.skyBox 获取默认的天空盒子的背景图。

重新设置天空 ​

这里重新指定的背景图片

  • 每个面都必须具有相同的宽度和高度
js
  viewer.value.scene.skyBox = new Cesium.SkyBox({
    sources: {
        positiveX: wallImage,
        negativeX: wallImage,
        positiveY: wallImage,
        negativeY: wallImage,
        positiveZ: wallImage,
        negativeZ: wallImage,
    }
})
